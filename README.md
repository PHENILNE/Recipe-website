Omelette Recipe Website

A simple recipe website built using HTML and CSS.  
This project was one of my first web development projects and focuses on basic webpage structure and styling.

 Features:
- Clear recipe layout
- Ingredients and step-by-step instructions
- Clean and readable design
- Beginner-friendly UI

Technologies Used
- HTML – structure and content  
- CSS – styling and layout  

 How to Run the Project
-Open index.html on your browser.
